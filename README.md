# 💄 Khawahish Beauty Makeover

A modern, responsive, and elegant beauty salon website built with **React.js** for **Khawahish Beauty Makeover**.

🌐 **Live Website:** https://your-vercel-link.vercel.app

---

## ✨ Features

- 💄 Beautiful Hero Section
- 👩 About Us
- ⭐ Why Choose Us
- 💅 Services Section
- 🖼️ Filterable Gallery
- 🔍 Image Lightbox
- 💬 Customer Testimonials
- 📅 WhatsApp Appointment Booking
- 📱 Fully Responsive Design
- 📞 Contact Section
- ⬆️ Scroll-to-Top Button
- 💚 Floating WhatsApp Button
- ✨ Scroll Animations (AOS)
- 🎯 Active Navigation Highlight
- 🚀 Deployed on Vercel

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Khawahish-Beauty-Makeover.git
```

Go inside the project

```bash
cd Khawahish-Beauty-Makeover
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
